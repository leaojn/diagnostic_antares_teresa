# Quiz Diagnóstico Estratégico

Aplicação React (Vite) para coleta de respostas de um questionário sobre compras e operação, com envio das respostas ao Supabase.

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Configure as variáveis `VITE_SUPABASE_URL` e `VITE_SUPABASE_ANON_KEY` (veja `.env.example`).
